# DataMatcher Pro

Zaawansowane narzędzie do łączenia danych z plików Excel na podstawie kluczy (MDM, EAN, SKU).

## Instalacja

```bash
pip install -r requirements.txt
```

## Uruchomienie

```bash
python main.py
```

## Funkcjonalności

- Łączenie danych z wielu źródeł (Excel, CSV)
- Elastyczne mapowanie kolumn z różnymi trybami zapisu
- Podgląd zmian z kolorowaniem
- Profile mapowań do wielokrotnego użycia
- Automatyczne raporty wykonania

## Wymagania

- Python 3.10+
- pandas, openpyxl, ttkbootstrap
